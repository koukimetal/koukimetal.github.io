jsTree Everywhere

How to use

Right click on "root" and Create . Create directory structures. Finally, click Publish . Get a snippet and put it on your website.

Requirements to use snippets

head tag has to include following 3 lines. 
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jstree/3.2.1/themes/default/style.min.css" />
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/1.12.1/jquery.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/jstree/3.2.1/jstree.min.js"></script>
If you want to use icons (e.g. ), head tag has to include following line. 
<link href="https://cdnjs.cloudflare.com/ajax/libs/simple-line-icons/2.3.0/css/simple-line-icons.css" rel="stylesheet">

ID

You can give an id to a published tree. Otherwise id should be tree_<random number>.
