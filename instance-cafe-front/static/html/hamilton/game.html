HamiltonCycle: You should find a cycle which contains all vertices.
Double click!, then you can select a vertex. It should become yellow.
When you selected a vertex again,
if the previous selected vertex has an edge to the selected vertex,
we recognize that you want to use the edge to make hamilton cycle.
If the edge was already used, the edge should be canceled.
You can cancel the selected vertex, when you double clicked it again.
